# 💰 AI Smart Expense Tracker(LaDu

A smart AI-powered expense tracking application built with **React + Vite**, designed to make managing personal finances as simple as speaking or scanning receipts.

---

## 🚀 Features

- 🎤 Voice-based expense input (speech → text)
- 📷 Camera receipt scanning
- 🤖 AI-powered transaction parsing
- 💸 Auto categorization (Food, Transport, Income, etc.)
- 📊 Transaction history & filtering
- 🌍 Multi-language support (Arabic / English)
- 📱 Mobile-friendly UI
- 🔗 Works with ngrok for real device testing

---

## 🧠 AI Concept

Instead of manually typing expenses, users can:

- Say:  
  *"I spent 20 dollars on pizza today"*
- Or scan a receipt using the camera

The system automatically:
- Extracts data
- Categorizes the expense
- Saves it in the transactions list
